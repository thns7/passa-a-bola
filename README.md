# Passa Bola App

Web app feito com **Next.js**, **React** e **Tailwind CSS** para acompanhar o **futebol feminino**: notícias, eventos, calendário, perfis de jogadoras e posts.

---

## Tecnologias

- Next.js  
- React  
- Tailwind CSS  
- Lucide Icons
